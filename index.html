CesiumExt 1 — JavaScript Toolkit for building world-class 3D globes and maps

CesiumExt is Open Source and enables building web applications with world-class 3D globes and maps. It is a JavaScript framework that is built based on CesiumJS and interface of the ExtJS library.

CesiumExt 1 is the first base version which uses the versions ExtJS 6.2.0 and CesiumJS 1.61

About CesiumExt

This framework helps GIS developers to design, develop, test applications in a rapid and more efficient way

Download & how to use in your application

There are multiple ways to download and use GeoExt.

  • using npm: npm install @CesiumExt/cesiumext
  • by cloning with git: git clone https://github.com/CesiumExt/cesiumext1.git
  • downloading the latest official release
  • downloading the latest raw source code

Versions of GeoExt

The following table lists the versions of OpenLayers and ExtJS that the respective version of GeoExt is compatible with:

OpenLayers ExtJS GeoExt status
4.4.x 6.2.0 master work in progress
4.3.3 6.2.0 3.2.0 released
4.3.3 6.2.0 3.1.0 released
3.20.1 6.2.0 3.0.0 released
2.13.1 5.1.0 / 4.2.1 2.1.x released
2.11.1 3.4.0 1.1.x released

Of course you can still use GeoExt 1.x or GeoExt 2.x, but these libraries do not receive much attention these days.
